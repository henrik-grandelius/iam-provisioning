# IAM Provisioning
Ett enkelt pytonprogram med inspiration från [IAM.Provisioning.Client
Public](https://github.com/WILGROSS/IAM.Provisioning.Client/tree/main). 

## Hur kör jag igång det?

Börja med att [installera pyton](https://www.python.org/downloads/). Efter installation, klona repot, skapa en virtuell miljö, installera beroenden och kör!  

Klona repot, öppna mappen med VS Code eller annan IDE, starta en terminal och kör följande kommandon:

```
# Skapa virtuell miljö
python -m venv venv

# Aktivera virtuell miljö
.\venv\Scripts\activate

# Installera requirements.txt
pip install -r requirements.txt

# Starta appen
py .\provision_users.py
```
